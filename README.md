## ❌⭕ Tic-Tac-Toe Game

A simple, interactive Tic-Tac-Toe game that allows two players to compete locally or play solo for quick practice and fun.

## 🎮 Overview

This Tic-Tac-Toe game is a browser-based implementation of the classic 3x3 grid game. Players take turns marking Xs and Os by clicking on the board. The game detects wins, draws, and supports quick resets for continuous play.

## ✨ Features
- 👥 Two-player mode (local turn-based play)
- 🤖 Optional solo play for self-practice
- 🖱️ Click-to-play intuitive interface
- 🏆 Automatic win and draw detection
- 🔄 Reset button to restart the game instantly
- 🎨 Clean and minimal UI for easy gameplay

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript
